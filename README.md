# Power

> A fitness app for women :muscle:    
### :anatomical_heart: [Presentation Link](https://docs.google.com/presentation/d/14MPhbMzKi-TRfgfjAky1mLuzObKzVVHIdB9asZNYfng/edit?usp=sharing)

## Features
- Simple home screen:
  - Time how long you work out
  - Track water intake
  - Track calories
- Side menu:
  - Exercise - How To's
  - Workout Routines
  - Nutrition
  - Mental Health
  - Levels
